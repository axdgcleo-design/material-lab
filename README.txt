★★★ UPGRADING FROM v0.1 / v0.2 ★★★
----------------------------------------------------------------
DO NOT delete your old folder! The "data" folder contains
all your materials.

How to upgrade:
1. Extract new zip to a TEMP location
2. Copy these to your OLD folder (overwrite):
   - START.bat
   - app/  (whole folder)
   - README.txt
3. DO NOT touch the "data" folder
4. Double-click START.bat


FIRST TIME SETUP
----------------------------------------------------------------
1. Install Python 3.9+: https://www.python.org/downloads/
   *** CHECK "Add Python to PATH" during install ***

2. Double-click START.bat
