# 未來計畫

## 即將做(優先順序高)

### v0.4 — 估價單系統
**為什麼做**: 案場已經有材料 + 數量,自動算總價最自然
- 進入案場 → 「產生估價單」按鈕
- 自動計算:材料費小計、含工費小計、總計
- 可填:工程費、設計費、雜支
- 一鍵輸出列印版 HTML(類似業主版材料書的設計)
- 可選:含工 vs 不含工兩種版本

### v0.5 — Excel 匯入匯出
**為什麼做**: 資料量大時管理更彈性,也方便給會計、給工班
- 材料庫匯出 Excel(可選欄位、可選分類)
- 案場材料清單匯出 Excel
- Excel 匯入材料(從現有 Excel 庫快速搬進系統)
- 模板下載:給使用者一個標準範本

### v0.6 — 案場照片整合
**為什麼做**: 業主版材料書要有實景照才更完整
- 案場新增「實景照」上傳區
- 每張實景照可標註「這是用了 X 材料」
- 業主版材料書自動把實景照配對到材料

## 中期計畫

### v0.7 — 廠商管理
- 獨立的廠商資料庫(目前是文字欄位)
- 一個廠商可關聯多個材料
- 廠商頁面:看這個廠商有哪些材料、總採購金額
- 廠商評分、合作紀錄

### v0.8 — 多設計師協作
**前提**: 漣一團隊擴大
- 多使用者帳號
- 權限分級(主設計師 / 助理 / 工地)
- 材料庫共用,案場可指定負責人
- 改用網路版部署(內部 NAS 或私人雲)

### v0.9 — 提案系統
- 設計師端:整理材料給業主初次提案
- 業主端:看到「方案 A / B / C」,可給意見
- 連結到案場,確認後變成正式材料清單
